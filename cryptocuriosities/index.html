What about crypto?

I am curious about the rapid developments in the crypto domain. But I think that there is more in crypto assets than just trading tokens and NFT. I tend to think that there is a possibility for crypto-assets to change the model of out socities in the decades to come.

So, in a sense, I may be considered as a crypto-enthusiast ! Thus, I started getting interested in the subject and wanted to learn with practice. I put hands and learned some topics through some projects ! Please note that, since those projects have only an educational objective, they have not been maintained and their respective code should not be used...

Smart contracts on different blockchains

The WavePortal: an Ethereum smart contract

The WavePortal is kind of a Web3 version of Twitter. It is a place where anyone can say hello and send a message that will be not only displayed on the page but also saved on the Ethereum blockchain. This WavePortal is actually my very first experience of learning while building and deploying an Ethereum smart contract as well as the associated website. So what did I achieve during this project?:
  • I wrote a smart contract in Solidity, compiled it locally and ran it.
  • I stored data on the blockchain.
  • I setted up a basic react app.
  • I deployed the smart contract to a real testnet.
  • I made it possible to connect a wallet to the app to call the smart contract.

The GifPortal: building a Dapp on Solana

Another portal?? Well... OK. This GifPortal project is definitly a way to learn something bigger than the portal-style application: Solana ! Solana is usually known as a super-fast and low-gas fee blockchain. For these reasons, it has been used in many NFT-based applications. Here are some of the concepts that were addressed during this project:
  • Writting my first Solana program.
  • Store data on the program.
  • Deploy the program to the devnet and call it from web app.
  • Give the possibility for visitors to submit GIFs to the Solana program.
  • Building a button for the user to connect to the web app with Phantom Wallet.

NFTs related projects

My NFT Collection

The aim of this NFT Collection project is to let users connect their Ethereum wallet to the Web3 app and mint an NFT that they will own. This project is not focusing on the graphical aspect of the NFT. Thus, NFTs are actually composed of words. This was a good way to get some experience and learn some things:
  • What does it mean to mint an NFT ? How does it work ?
  • How to send NFT on an Ethereum wallet.
  • How to visualize NFT on OpenSea.

A Turn-based NFT game

One of the popular use of NFTs lately has been gaming. This NFT game project gave me the opportunity to further some of the concepts I've seen in previous projects with a little touch of this gaming universe. To be honest, the main objective of this project was to have fun. But it also let me approach some interesting points such as:
  • Thinking about the game logic and building NFT characters.
  • Holding dynamic data on an NFT.
  • Discovering Events (signals that can be fired and capture by other parts of the code to detect an event and act accordingly)
  • Diving deeper in the creation of the web app.
